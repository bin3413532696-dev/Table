## 1. 项目概述

个人工作站是一个**中文界面的单页应用**（SPA），面向个人效率场景。所有数据存储于浏览器 `localStorage`，并自动同步到项目 `data/` 目录（通过 Webpack Dev Server Middleware），**无独立后端服务**。应用基于 HashRouter 实现 6 个功能页面，共享统一的侧边栏 + 顶栏布局。

### 核心功能模块

| 模块 | 说明 |
|------|------|
| **仪表盘** | 聚合任务、财务数据的首页 |
| **任务** | 待办事项管理，优先级和截止日期 |
| **财务** | 收支记录与统计（面向 LLM API 费用场景） |
| **知识库** | 笔记管理、语义搜索、AI 对话、知识图谱 |
| **工具** | 计算器、取色器、JSON 格式化工具 |
| **设置** | 个人资料，安全隐私、数据导入导出、主题切换 |

## 4. 构建配置

### Webpack (webpack.config.js)

| 配置项 | 值 |
|--------|-----|
| Entry | `./src/index.tsx` |
| Output | `dist/bundle.js` + chunk files |
| 默认端口 | 3266 |
| HTML 模板 | `./index.html` |
| Loader | babel-loader (.ts/.tsx/.js/.jsx) |
| CSS 链 | style-loader → css-loader → postcss-loader |
| historyApiFallback | 已启用，带 `_p/\d+/` 重写规则 |
| 代码分割 | React.lazy + Suspense，按页面拆分 |

### Transformers.js Webpack 配置

```javascript
resolve: {
  extensions: ['.ts', '.tsx', '.js', '.jsx'],
  alias: {
    '@huggingface/transformers': path.resolve(__dirname, 'node_modules/@huggingface/transformers/dist/transformers.web.js'),
    'sharp$': false,
    'onnxruntime-node$': false,
    'onnxruntime-web$': false,
  },
  fallback: {
    fs: false,
    path: false,
    url: false,
  }
}
```

### 代码分割策略

页面组件使用 `React.lazy()` 动态导入，实现按需加载：

```typescript
const Dashboard = lazy(() => import('./pages/Dashboard'));
const Tasks = lazy(() => import('./pages/Tasks'));
const Finance = lazy(() => import('./pages/Finance'));
const Knowledge = lazy(() => import('./pages/Knowledge'));
const Tools = lazy(() => import('./pages/Tools'));
const Settings = lazy(() => import('./pages/Settings'));
```

构建产物：
- `bundle.js` (~370 KiB) — 主入口
- `[hash].bundle.js` (~383 KiB) — 第三方库 vendors
- `[hash].bundle.js` (~58 KiB) — Knowledge
- `[hash].bundle.js` (~38 KiB) — Tools
- `[hash].bundle.js` (~29 KiB) — Finance
- `[hash].bundle.js` (~24 KiB) — Tasks
- `[hash].bundle.js` (~15 KiB) — Dashboard

## 13. 常见开发任务

2. 使用 `Node.create()` 或 `Extension.create()`
