## 1. 项目概述

个人工作站是一个**中文界面的单页应用**（SPA），面向个人效率场景。所有数据存储于浏览器本地存储（localStorage + IndexedDB），并通过双向同步机制与项目 `data/` 目录保持一致，**无独立后端服务**。应用基于 HashRouter 实现 5 个功能页面，共享统一的侧边栏 + 顶栏布局。

### 核心功能模块

| 模块 | 说明 |
|------|------|
| **仪表盘** | 聚合任务、财务数据的首页 |
| **任务** | 待办事项管理，优先级和截止日期 |
| **财务** | 收支记录与统计（面向 LLM API 费用场景） |
| **工具** | 计算器、取色器、JSON 格式化工具 |
| **设置** | 个人资料，安全隐私、数据导入导出、主题切换 |
| **智能助手** | 全局 AI 智能体（Ctrl+K 呼起） |

## 5. 数据层

**特性**:
- 自定义 `Store<T>` 类封装 CRUD
- 数据验证器 `isValidFinanceRecord`, `isValidTask`

- 发布订阅 `subscribe(listener)`
- 自动同步到后端 API

### 5.2 全局状态管理

**文件**: `src/store/useStore.tsx`

**状态结构**:
```typescript
interface AppState {
  finance: FinanceRecord[];
  tasks: Task[];
  loading: boolean;
}
```

**Hooks**:
- `useStore()` — 完整状态
- `useFinance()` — 财务数据 + 操作
- `useTasks()` — 任务数据 + 操作

## 9. 通用智能体

### 9.1 概述

通用智能体是一个全局 AI 助手，可通过 `Ctrl+K` 快捷键呼起，支持跨模块数据查询和操作。

### 9.2 架构

```
src/agent/
├── types.ts           # 类型定义
├── AgentEngine.ts     # 智能体引擎
├── AgentContext.tsx   # 状态管理
└── tools/
    └── index.ts       # 工具注册
```

### 9.3 工具系统

| 工具类型 | 工具名称 | 需确认 |
|----------|----------|--------|
| query | query_finance, get_finance_stats | No |
| query | query_tasks, get_task_stats | No |
| query | get_overview, cross_module_analysis | No |
| mutation | add_finance_record, delete_finance_record | Yes |
| mutation | create_task, update_task, delete_task | Yes |

### 9.4 使用方式

```typescript

Ctrl + K

// 对话示例
"查询我的财务状况"
"创建一个任务：完成报告"
"分析本月任务完成率与支出关联"
```

## 10. 数据同步机制

### 10.1 双向同步

```
┌─────────────────┐                    ┌─────────────────┐
│   浏览器页面     │                    │   data 目录      │
└────────┬────────┘                    └────────┬────────┘
         │                                      │
         │  修改数据 → POST /api/sync-data      │
         │ ─────────────────────────────────────│
         │                                      │ 写入文件
         │                                      │
         │                                      │ 文件变化
         │  WebSocket: data-changed             │
         │ ─────────────────────────────────────│
         │  GET /api/load-data                  │
         │ ─────────────────────────────────────│
         │  更新 localStorage                   │
```

### 10.2 同步服务

**文件**: `src/lib/dataSync.ts`

```typescript
// 初始化加载
initializeData()

// 启动自动同步（页面→文件）
startAutoSync()

// 启动实时同步（文件→页面）
startRealtimeSync()

// 手动同步
syncAll()
syncFinance()
syncTasks()
```

### 10.3 data 目录规范

```
data/
├── README.md            # 目录说明
├── config.json          # 全局配置
├── finance/
│   └── finance.json     # 财务记录数组
├── tasks/
│   └── tasks.json       # 任务记录数组
├── folders/
│   └── folders.json     # 文件夹结构（预留）
└── backup/
    └── *.json           # 备份文件
```
