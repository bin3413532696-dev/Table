## 1. 项目概述

个人工作站是一个**中文界面的单页应用**（SPA），面向个人效率场景。所有数据存储于浏览器本地存储（localStorage + IndexedDB），并通过双向同步机制与项目 `data/` 目录保持一致，**无独立后端服务**。应用基于 HashRouter 实现 6 个功能页面，共享统一的侧边栏 + 顶栏布局。

### 核心功能模块

| 模块 | 说明 |
|------|------|
| **仪表盘** | 聚合任务、财务数据的首页 |
| **任务** | 待办事项管理，优先级和截止日期 |
| **财务** | 收支记录与统计（面向 LLM API 费用场景） |
| **知识库** | 笔记管理、语义搜索、AI 对话、知识图谱 |
| **工具** | 计算器、取色器、JSON 格式化工具 |
| **设置** | 个人资料，安全隐私、数据导入导出、主题切换 |
| **智能助手** | 全局 AI 智能体（Ctrl+K 呼起） |

## 5. 数据层

### 5.1 localStorage 数据 (finance/tasks)

**文件**: `src/db/index.ts`

**特性**:
- 自定义 `Store<T>` 类封装 CRUD
- 数据验证器 `isValidFinanceRecord`, `isValidTask`
- 事务支持 `runInTransaction`
- 发布订阅 `subscribe(listener)`
- 自动同步到后端 API

### 5.2 IndexedDB 数据 (notes)

**文件**: `src/db/knowledge.ts`

```typescript
interface KnowledgeNote {
  id: string;
  title: string;
  content: string;
  tags: string[];
  links: string[];
  backlinks: string[];
  createdAt: number;
  updatedAt: number;
}
```

**API**: `noteOperations`
- `create(note)` → `Promise<string>`
- `update(id, updates)` → `Promise<void>`
- `delete(id)` → `Promise<void>`
- `getAll()` → `Promise<KnowledgeNote[]>`
- `getById(id)` → `Promise<KnowledgeNote | undefined>`
- `getCount()` → `Promise<number>`
- `findBacklinks(targetId)` → `Promise<KnowledgeNote[]>`
- `updateContent(id, content, title?)` → `Promise<void>` — 自动提取 WikiLinks
- `importNotes(notes)` → `Promise<{ success, imported }>`
- `clearAll()` → `Promise<void>`

**订阅机制**: `subscribeKnowledge(listener)` — 数据变化通知 UI

**统计**: `knowledgeStats.getStats()`, `knowledgeStats.getTagStats()`

### 5.3 全局状态管理

**文件**: `src/store/useStore.tsx`

**状态结构**:
```typescript
interface AppState {
  finance: FinanceRecord[];
  tasks: Task[];
  notes: KnowledgeNote[];  // 新增
  loading: boolean;
}
```

**Hooks**:
- `useStore()` — 完整状态
- `useFinance()` — 财务数据 + 操作
- `useTasks()` — 任务数据 + 操作
- `useNotes()` — 笔记数据 + 操作（新增）

## 11. 数据同步机制

### 11.1 双向同步

```
┌─────────────────┐                    ┌─────────────────┐
│   浏览器页面     │                    │   data 目录      │
└────────┬────────┘                    └────────┬────────┘
         │                                      │
         │  修改数据 → POST /api/sync-data      │
         │ ─────────────────────────────────────│
         │                                      │ 写入文件
         │                                      │
         │                                      │ 文件变化
         │  WebSocket: data-changed             │
         │ ─────────────────────────────────────│
         │  GET /api/load-data                  │
         │ ─────────────────────────────────────│
         │  更新 IndexedDB/localStorage         │
```

### 11.2 同步服务

**文件**: `src/lib/dataSync.ts`

```typescript
// 初始化加载
initializeData()

// 启动自动同步（页面→文件）
startAutoSync()

// 启动实时同步（文件→页面）
startRealtimeSync()

// 手动同步
syncAll()
syncFinance()
syncTasks()
syncNotes()
```
